**၄။ သဒ္ဒါရှင်းလင်းချက် (Grammar Explanations)**

**၁. ここ / そこ / あそこ / こちら / そちら / あちら**
*   `ここ`၊ `そこ`၊ `あそこ` သည် နေရာကိုညွှန်ပြသည်။ `ここ` သည် ပြောသူရှိသောနေရာ၊ `そこ` သည် နားထောင်သူရှိသောနေရာ၊ `あそこ` သည် ထိုနှစ်ဦးစလုံးနှင့် ဝေးကွာသောနေရာကို ညွှန်ပြသည်။
*   `こちら`၊ `そちら`၊ `あちら` သည် နေရာအရပ်ကို ညွှန်ပြပြီး `ここ`၊ `そこ`၊ `あそこ` ၏ အစားထိုးအဖြစ် မျက်မှောက်ရှိ မျက်နှာမူရာနေရာကို ညွှန်ပြရာတွင်လည်း အသုံးပြုသည်။ ဤအသုံးအနှုန်းသည် ပို၍ ယဉ်ကျေးသော စကားလုံးကို ဖော်ပြသည်။
*   [မှတ်ချက်] ပြောသူသည် နားသူအား မိမိ၏နယ်ပယ်အတွင်း ရှိသည်ဟု သတ်မှတ်ပါက နှစ်ဦးရှိသောနေရာကို `ここ` ၊ အနည်းငယ်ဝေးသောနေရာကို `そこ` ၊ အဝေးဆုံးနေရာကို `あそこ` ဟု ပြောသည်။

**၂. N は နေရာ です**
ဤဝါကျပုံစံကိုအသုံးပြု၍ အရာဝတ္ထု၊ နေရာ၊ လူပုဂ္ဂိုလ်စသည်တို့ တည်ရှိသောနေရာကို ပြောဆိုဖော်ပြသည်။
*   ① お手洗い は あそこ です。 (အိမ်သာက ဟိုမှာပါ။)
*   ② 電話 は 2階 です。 (တယ်လီဖုန်းက ၂ထပ်မှာပါ။)
*   ③ 山田さん は 事務所 です。 (မစ္စတာယာမာဒါက ရုံးခန်းမှာပါ။)

**၃. どこ / どちら**
`どこ` သည် နေရာကို မေးမြန်းသော အမေးစကားလုံးဖြစ်ပြီး၊ `どちら` သည် ရှေ့ရှုရာအရပ်ကို မေးမြန်းသော အမေးစကားလုံး ဖြစ်သည်။ နေရာကို မေးမြန်းရာတွင် `どちら` ကို အသုံးပြုပါက ပို၍ ယဉ်ကျေးသော အမေးစကားလုံး ဖြစ်သည်။
*   ④ お手洗い は どこ ですか。 (အိမ်သာက ဘယ်မှာပါလဲ။)
    ...... あそこ です。 (......ဟိုဘက်မှာပါ။)
*   ⑤ エレベーター は どちら ですか。 (ဓာတ်လှေကားက ဘယ်မှာပါလဲ။)
    ...... あちら です。 (......ဟိုဘက်မှာပါ။)

တစ်ဖန် နိုင်ငံ၊ ကုမ္ပဏီ၊ ကျောင်းစသည့် မိမိ၏ နာမည်စာရင်းရှိရာ နေရာနှင့် အသင်းအဖွဲ့အမည်များကို မေးမြန်းရာတွင်လည်း အမေးစကားလုံးတွင် `なん` ကို အသုံးမပြုဘဲ၊ `どこ` ၊ `どちら` ကို အသုံးပြုသည်။ `どこ` ထက် `どちら` ကို ပို၍ ယဉ်ကျေးရာတွင် အသုံးပြုသည်။
*   ⑥ 学校 は どこ ですか。 (ဘယ်ကျောင်းကပါလဲ။)
*   ⑦ 会社 は どちら ですか。 (ဘယ်ကုမ္ပဏီကပါလဲ။)

**၄. N1 の N2**
N1 သည် နိုင်ငံအမည်ဖြစ်ပြီး၊ N2 သည် ထုတ်ကုန်ပစ္စည်းဖြစ်ပါက `の` သည် ထိုနိုင်ငံထုတ်ကုန်ပစ္စည်းဟု အဓိပ္ပာယ်ရသည်။ N1 သည် ကုမ္ပဏီအမည်ဖြစ်ပြီး၊ N2 သည် ထုတ်ကုန်ပစ္စည်းဖြစ်ပါက N1 `の` သည် ထိုကုမ္ပဏီမှထုတ်လုပ်သော ကုန်ပစ္စည်းဟု အဓိပ္ပာယ်ရသည်။ မည်သည့်နိုင်ငံမှ၊ မည်သည့် ကုမ္ပဏီမှထုတ်လုပ်သည်ကို မေးမြန်းလိုပါက `どこ` ကို အသုံးပြုသည်။
*   ⑧ これ は どこ の コンピューター ですか。 (ဒါက ဘယ်က ကွန်ပျူတာပါလဲ။)
    ...... 日本 の コンピューター です。 (......ဂျပန်က ကွန်ပျူတာပါ။)
    ...... パワー電気 の コンピューター です。 (......ပါဝါလျှပ်စစ်က ကွန်ပျူတာပါ။)

**၅. こ／そ／あ／ど (အညွှန်းစကားလုံး) ဇယား**

| | こ အုပ်စု | そ အုပ်စု | あ အုပ်စု | ど အုပ်စု |
| :--- | :--- | :--- | :--- | :--- |
| **အရာဝတ္ထု** | これ | それ | あれ | どれ (သင်ခန်းစာ-၁၆) |
| **အရာဝတ္ထု/လူပုဂ္ဂိုလ်** | この N | その N | あの N | どの N (သင်ခန်းစာ-၁၆) |
| **နေရာ** | ここ | そこ | あそこ | どこ |
| **ရှေးရှုရာ / နေရာ (ယဉ်ကျေး)** | こちら | そちら | あちら | どちら |

**၆. お～**
နားသူ သို့မဟုတ် တတိယလူနှင့် သက်ဆိုင်သောအကြောင်းအရာ၌ တွဲဖက်လေ့ရှိသော ရှေ့ဆက်စကားလုံး `お` သည် ပြောသူ၏ ရိုသေလေးစားမှုကို ဖော်ပြသည်။
*   ⑨ [お]国 は どちら ですか。 (ဘယ်နိုင်ငံကပါလဲ။)
