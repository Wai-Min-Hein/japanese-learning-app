### **Chapter 7.txt**

**ဝေါဟာရများ (Vocabulary)** 

* **きります (切ります)** : ဖြတ်သည်၊ ညှပ်သည်၊ လှီးသည် 


* 
**あげます** : ပေးသည် 


* 
**もらいます** : လက်ခံရယူသည် 


* 
**かします (貸します)** : ချေးငှားသည် (ချေးပေးသည်) 


* 
**かります (借ります)** : ငှားသည် (ချေးယူသည်) 


* 
**おしえます (教えます)** : သင်သည် (သင်ပေးသည်) 


* 
**ならいます (習います)** : သင်သည် (သင်ယူသည်) 


* 
**はし** : တူ (စားပွဲတင်တူ) 


* 
**けしゴム (消しゴム)** : ခဲဖျက် 


* 
**はな (花)** : ပန်း 


* 
**おかね (お金)** : ပိုက်ဆံ 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. 
**N (ကိရိယာ) で V** : အသုံးပြုသည့် ကိရိယာ သို့မဟုတ် နည်းလမ်းကို ပြသည် (ဥပမာ- တူနဲ့ စားသည်) ။


2. 
**"စာလုံး/ဝါကျ" は ~語で 何ですか** : တခြားဘာသာစကားဖြင့် မည်သို့ခေါ်သနည်းဟု မေးမြန်းရာတွင် သုံးသည် ။


3. 
**N1 (လူ) に N2 ကို あげます** : N1 အား N2 ကို ပေးသည် ။


4. 
**N1 (လူ) に N2 ကို もらいます** : N1 ထံမှ N2 ကို ရရှိသည် ။


5. 
**もう Vました** : လုပ်ဆောင်မှု ပြီးမြောက်သွားပြီဟု အဓိပ္ပာယ်ရသည် ။

### **Chapter 8.txt**

**ဝေါဟာရများ (Vocabulary)** 

* 
**ハンサム [な]** : ချောမောသော 


* **きれい [な]** : လှပသော၊ သန့်ရှင်းသော 


* 
**しずか [な] (静か)** : တိတ်ဆိတ်သော 


* 
**ゆうめい [な] (有名)** : နာမည်ကြီးသော 


* 
**しんせつ [な] (親切)** : ကြင်နာတတ်သော 


* 
**おおきい (大きい)** : ကြီးမားသော 


* 
**あたらしい (新しい)** : သစ်လွင်သော 


* 
**あつい (暑い/熱い)** : ပူသော 


* 
**むずかしい (難しい)** : ခက်ခဲသော 


* **たかい (高い)** : ဈေးကြီးသော၊ မြင့်သော 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. 
**နာမဝိသေသန (Adjectives)** : 'န' (na) နှင့် 'အိ' (i) ဟူ၍ နှစ်မျိုးရှိသည် ။


2. 
**N は Adj です** : နာမ်၏ အခြေအနေကို ဖော်ပြသည် ။


3. 
**Adj-na な N / Adj-i N** : နာမ်ကို ရှေ့မှနေ၍ အထူးပြုသည် ။


4. 
**とても / あまり** : "အလွန်" နှင့် "သိပ်" (အငြင်းဝါကျတွင် သုံးသည်) ဟု အဓိပ္ပာယ်ရသည် ။


5. 
**N は どうですか** : ဘယ်လိုနေလဲဟု သဘောထားကို မေးမြန်းသည် ။

### **Chapter 9.txt**

**ဝေါဟာရများ (Vocabulary)** 

* 
**わかります** : နားလည်သည် 


* 
**あります** : ရှိသည် (ပိုင်ဆိုင်ခြင်း) 


* 
**すき [な] (好き)** : ကြိုက်သော 


* 
**じょうず [な] (上手)** : တော်သော 


* 
**りょうり (料理)** : ဟင်းလျာ 


* 
**じかん (時間)** : အချိန် 


* 
**ようじ (用事)** : အလုပ်ကိစ္စ 


* **やくそく (約束)** : ကတိ၊ ချိန်းဆိုချက် 


* 
**どうして** : ဘာကြောင့် 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. 
**N が あります/わかります** : "ရှိသည်" နှင့် "နားလည်သည်" တို့၏ ကံပုဒ်ကို 'ဂါ' (ga) ဝိဘတ်ဖြင့် ပြသည် ။


2. 
**N が 好き/上手 です** : နှစ်သက်မှုနှင့် ကျွမ်းကျင်မှုကို 'ဂါ' (ga) ဖြင့် ပြသည် ။


3. **よく / だいたい / たက်စန်** : "ကောင်းကောင်း"၊ "အကြမ်းဖျင်း"၊ "အများကြီး" ဟု အတိုင်းအတာပြသည် ။


4. 
**S1 から S2** : S1 ကြောင့် S2 ဖြစ်သည် (အကြောင်းပြချက်ကို ပြသည်) ။

### **Chapter 10.txt**

**ဝေါဟာရများ (Vocabulary)** 

* 
**あります / います** : ရှိသည် (သက်မဲ့ / သက်ရှိ) 


* 
**うえ (上) / した (下)** : အပေါ် / အောက် 


* 
**まえ (前) / うしろ (後ろ)** : ရှေ့ / နောက် 


* 
**みぎ (右) / ひだり (左)** : ညာ / ဘယ် 


* 
**なか (中) / そと (外)** : အထဲ / အပြင် 


* 
**となり (隣)** : ဘေးနား 


* 
**あいだ (間)** : အကြား 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. 
**N が あります/います** : တည်ရှိမှုကို ဖော်ပြသည် ။


2. 
**နေရာ に N が あります/います** : တစ်စုံတစ်ခုရှိရာ နေရာကို ဖော်ပြသည် ။


3. 
**N は နေရာ に あります/います** : တစ်စုံတစ်ခု တည်ရှိရာ နေရာကို အဓိကထား ပြောဆိုသည် ။


4. 
**N1 の N2 (တည်နေရာ)** : တည်နေရာ ဆက်စပ်မှုကို ပြသည် (ဥပမာ- စားပွဲပေါ်) ။

### **Chapter 11.txt**

**ဝေါဟာရများ (Vocabulary)** 

* 
**ひとつ (1つ) ~ とお (10)** : အရာဝတ္ထု ရေတွက်ပုံ ၁ မှ ၁၀ 


* 
**~にん (人)** : -ယောက် 


* 
**~だい (台)** : -စီး (စက်ပစ္စည်း) 


* 
**~まい (枚)** : -ချပ်/ထည် (ပါးလွှာသောအရာ) 


* 
**~かい (回)** : -ကြိမ် 


* 
**~じかん (時間)** : -နာရီကြာ 


* 
**~かげつ (か月)** : -လကြာ 


* 
**どのくらい** : ဘယ်လောက်ကြာကြာ 


* 
**~だけ** : -သာ/ပဲ 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. 
**အရေအတွက် အခေါ်အဝေါ်** : အရာဝတ္ထုအလိုက် ကွဲပြားသော မျိုးပြစကားလုံးများကို သုံးရသည် ။


2. 
**အရေအတွက် အသုံးပြုပုံ** : အခြေခံအားဖြင့် 'နာမ် + ဝိဘတ်' ၏ နောက်တွင် ထားသည် ။


3. 
**ကာလ に ~回 V** : သတ်မှတ်ကာလအတွင်း လုပ်ဆောင်သည့် အကြိမ်အရေအတွက်ကို ပြသည် ။


4. 
**N だけ** : ၎င်းထက်ပို၍ မရှိခြင်း သို့မဟုတ် ၎င်းမှအပ မရှိခြင်းကို ပြသည် ။

### **Chapter 12.txt**

**ဝေါဟာရများ (Vocabulary)** 

* 
**かんたん [な] (簡単)** : လွယ်ကူသော 


* 
**ちかい (近い) / とおい (遠い)** : နီးသော / ဝေးသော 


* 
**はやい (早い/速い) / おそい (遅い)** : မြန်သော / နှေးသော 


* 
**あたたかい (暖かい/温かい)** : နွေးသော 


* 
**すずしい (涼しい)** : အေးမြသော 


* 
**はる / なつ / あき / ふゆ** : နွေဦး / နွေ / ဆောင်းဦး / ဆောင်း 


* 
**どちら** : ဘယ်တစ်ခု (၂ခုမှ ၁ခုရွေးချယ်ခြင်း) 


* 
**いちばん** : အကြိုက်ဆုံး / ပထမဆုံး 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. **နာမ်/နာမဝိသေသန၏ အတိတ်ကာလ** : です -> でした ၊ じゃありません -> じゃありませんでした ဟု ပြောင်းလဲသည် ။


2. 
**N1 は N2 より Adj です** : N1 သည် N2 ထက် ပို၍ ~ ဖြစ်သည် ။


3. 
**N1 と N2 と どちらが Adj ですか** : N1 နှင့် N2 ဘယ်ဟာ ပို၍ ~ လဲ ။


4. 
**N [の]で 何/どこ... が いちばん Adj ですか** : တစ်ခုခုအတွင်း၌ အ ~ ဆုံးဖြစ်သည်ကို မေးမြန်းသည် ။
