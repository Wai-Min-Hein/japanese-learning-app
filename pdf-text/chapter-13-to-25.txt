### **Chapter 13.txt**

**ဝေါဟာရများ (Vocabulary)**

* **あそびます (遊びます)** : ကစားသည်၊ လည်ပတ်အပန်းဖြေသည်
* **およぎます (泳ぎます)** : ရေကူးသည်
* **むかえます (迎えます)** : သွားကြိုသည်
* **つかれます (疲れます)** : ပင်ပန်းသည်
* **だします [てがみを~] (出します)** : ပို့သည် [စာကို~]
* **はいります [きっさてんに~] (入ります)** : ဝင်သည် [လက်ဖက်ရည်ဆိုင်သို့~]
* **でます [きっさてんを~] (出ます)** : ထွက်သည် [လက်ဖက်ရည်ဆိုင်မှ~]
* **けっこんします (結婚します)** : လက်ထပ်သည်
* **かいものします (買い物します)** : ဈေးဝယ်သည်
* **しょくじします (食事します)** : ထမင်းစားသည်
* **さんぽします [こうえんを~] (散歩します)** : လမ်းလျှောက်သည် [ပန်းခြံထဲတွင်~]
* **ほしい (欲しい)** : လိုချင်သော
* **たいへん [な] (大変)** : ပင်ပန်းဆင်းရဲသော

**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)**

1. 
**N が 欲しいです** : တစ်စုံတစ်ခုကို လိုချင်ကြောင်း ဖော်ပြသည် ။


2. 
**Vます-たいです** : တစ်စုံတစ်ခုကို လုပ်ချင်ကြောင်း ဖော်ပြသည် ။


3. 
**N (နေရာ) へ Vます- / N (နာမ်) に 行きます/来ます/帰ります** : တစ်စုံတစ်ခု ပြုလုပ်ရန် သွားသည်/လာသည်/ပြန်သည် ဟူသော ရည်ရွယ်ချက်ကို ဖော်ပြသည် ။

### **Chapter 14.txt**

**ဝေါဟာရများ (Vocabulary)**

* **つけます** : ဖွင့်သည် (မီး၊ စက်ပစ္စည်း)
* **けします (消します)** : ပိတ်သည်၊ ဖျက်သည်
* **あけます (開けます)** : ဖွင့်သည် (တံခါး၊ ပြတင်းပေါက်)
* **しめます (閉めます)** : ပိတ်သည် (တံခါး၊ ပြတင်းပေါက်)
* **いそぎます (急ぎます)** : လောသည်၊ အလျင်လိုသည်
* **まちます (待ちます)** : စောင့်သည်
* **とめます (止めます)** : ရပ်သည် (ကား)
* **まがります (曲がります)** : ကွေ့သည်
* **もちます (持ちます)** : ကိုင်သည်၊ ယူဆောင်သည်
* **てつだいます (手伝います)** : ကူညီသည်

**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)**

1. 
**ကြိယာအုပ်စုများ** : ဂျပန်ကြိယာများကို အုပ်စု ၃ စု ခွဲခြားထားသည် ။


2. 
**Vて ပုံစံ** : ကြိယာများကို 'တေ' ပုံစံ ပြောင်းလဲအသုံးပြုခြင်း ။


3. 
**Vてください** : တစ်စုံတစ်ခု ပြုလုပ်ရန် တောင်းဆိုခြင်း ။


4. 
**Vています** : လက်ရှိ လုပ်ဆောင်နေဆဲ အခြေအနေကို ပြသည် ။


5. 
**Vましょうか** : တစ်စုံတစ်ယောက်အား ကူညီရန် ကမ်းလှမ်းခြင်း ။

* **のりかえます (乗り換えます)** : (ယာဉ်) လဲစီးသည်
* **あびます [シャワーを~] (浴びます)** : ချိုးသည် [ရေချိုးသည်]

* **だします (出します)** : ထုတ်သည်၊ ပေးသည်
* **おろします [おかねを~] (下ろします)** : ထုတ်သည် [ပိုက်ဆံထုတ်သည်]
* **はいります [だいがくに~] (入ります)** : ဝင်သည် [တက္ကသိုလ်သို့~]
* **でます [だいがくကို~] (出ます)** : ထွက်သည် [ကျောင်းဆင်းသည်/ဘွဲ့ရသည်]

**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)**

**ဝါကျများ ချိတ်ဆက်နည်း** : ကြိယာဝါကျများကို 'တေ' ပုံစံဖြင့် ဆက်တိုက်ချိတ်ဆက်ခြင်း ။


2. 
**V1てから、V2** : V1 ပြီးမှ V2 ကို လုပ်ဆောင်သည် ။


3. 
**N1 は N2 が Adj** : တစ်စုံတစ်ယောက်၏ ခန္ဓာကိုယ်အစိတ်အပိုင်း သို့မဟုတ် ဂုဏ်သတ္တိကို ဖော်ပြသည် ။

4. 
**V1-အဘိဓာန်ပုံစံ / Nの / ကာလ まえに、V2** : V2 မတိုင်မီ V1 ကို လုပ်သည် ။
