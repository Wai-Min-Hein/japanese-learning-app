### **Chapter 2.txt**

**ဝေါဟာရများ (Vocabulary)** 

* これ : ဒါ (ပြောသူ၏အနီး၌ရှိသောအရာ) 


* それ : အဲဒါ (နာသူ၏အနီး၌ရှိသောအရာ) 


* あれ : ဟိုဟာ (ပြောသူ၊ နာသူတို့မှ ကွာလှမ်းသောနေရာ၌ရှိသောအရာ) 


* この ~ / その ~ / あの ~ : ဒီ ~ / အဲဒီ ~ / ဟို ~ 


* ほん (本) : စာအုပ် 


* じしょ (辞書) : အဘိဓာန် 


* ざっし (雑誌) : မဂ္ဂဇင်း 


* しんぶん (新聞) : သတင်းစာ 


* ノート : ဗလာစာအုပ် 


* かぎ : သော့ 


* とけい (時計) : နာရီ 


* かさ (傘) : ထီး 


* かばん : အိတ် 


* [お]みやげ : အပြန်လက်ဆောင် 


* そう : အဲဒီလို 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. 
**これ/それ/あれ** : အရာဝတ္ထုကို ညွှန်ပြသော နာမ်များ ဖြစ်သည်။ 


2. 
**このN / そのN / あのN** : နာမ်ကို အထူးပြုသောအခါ အသုံးပြုသည်။ 


3. 
**そうです** : နာမ်ဝါကျတွင် အဟုတ်ဖြစ်ကြောင်း ဖြေကြားရာတွင် သုံးသည်။ 


4. 
**N1 の N2** : ပိုင်ဆိုင်မှု (ဥပမာ- ကျွန်တော့်ရဲ့စာအုပ်) သို့မဟုတ် မည်သည့်အရာနှင့် ပတ်သက်ကြောင်း (ဥပမာ- ကွန်ပျူတာစာအုပ်) ကို ဖော်ပြသည်။ 


5. 
**そうですか** : သတင်းအချက်အလက်သစ်ကို သိရှိနားလည်ကြောင်း ပြသသည်။

### **Chapter 3.txt**

**ဝေါဟာရများ (Vocabulary)** 

* ここ / そこ / あそこ : ဒီမှာ / အဲဒီမှာ / ဟိုမှာ 


* どこ : ဘယ်မှာ 


* こちら / そちら / あちら / どちら : ဒီဘက် / အဲဒီဘက် / ဟိုဘက် / ဘယ်ဘက် (ယဉ်ကျေးသောအသုံး) 


* きょうしつ (教室) : စာသင်ခန်း 


* しょくどう (食堂) : ထမင်းစားဆောင် 


* じむしょ (事務所) : ရုံးခန်း 


* うけつけ (受付) : ဧည့်ကြိုကောင်တာ 


* へや (部屋) : အခန်း 


* トイレ (お手洗い) : အိမ်သာ (သန့်စင်ခန်း) 


* エレベーター / エスカレーター : ဓာတ်လှေကား / စက်လှေကား 


* [お]くに (国) : နိုင်ငံ 


* かいしゃ (会社) : ကုမ္ပဏီ 


* いくら : ဘယ်လောက် (ဈေးနှုန်း) 


* えん (円) : ယန်း 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. 
**ここ/そこ/あそこ** : နေရာကို ညွှန်ပြသည်။ 


2. 
**N は နေရာ です** : အရာဝတ္ထု သို့မဟုတ် လူ တည်ရှိရာနေရာကို ဖော်ပြသည်။ 


3. **どこ / どちら** : နေရာ သို့မဟုတ် နိုင်ငံ၊ ကုမ္ပဏီ၊ ကျောင်း စသည့် အသင်းအဖွဲ့အမည်ကို မေးမြန်းရာတွင် သုံးသည်။ 


4. 
**N1 の N2** : မည်သည့်နိုင်ငံထုတ် သို့မဟုတ် မည်သည့်ကုမ္ပဏီထုတ် ဖြစ်ကြောင်း ဖော်ပြသည်။

### **Chapter 4.txt**

**ဝေါဟာရများ (Vocabulary)** 

* おきます (起きます) : အိပ်ရာထသည် 


* ねます (寝ます) : အိပ်သည် 


* はたらきます (働きます) : အလုပ်လုပ်သည် 


* やすみます (休みます) : အနားယူသည် 


* べんきょうします (勉強します) : စာလေ့လာသည် 


* おわります (終わります) : ပြီးဆုံးသည် 


* ぎんこう (銀行) : ဘဏ် 


* ゆうびんきょく (郵便局) : စာတိုက် 


* としょかん (図書館) : စာကြည့်တိုက် 


* いま (今) : ယခု 


* ~じ (時) / ~ふん (分) : ~နာရီ / ~မိနစ် 


* なんじ / なんぷん : ဘယ်နှနာရီ / ဘယ်နှမိနစ် 


* あさ / ひる / ばん (よる) : မနက် / နေ့လယ် / ည 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. 
**今 ~時 ~分です** : လက်ရှိအချိန်ကို ဖော်ပြသည်။ 


2. **Vます / Vません / Vました / Vませんでした** : ကြိယာ၏ အဟုတ်၊ အငြင်းနှင့် အတိတ်ကာလ ပုံစံများ ဖြစ်သည်။ 


3. 
**N (အချိန်) に V** : အပြုအမူ လုပ်ဆောင်သော အချိန်ကို ညွှန်ပြသည်။ 


4. 
**N1 から N2 まで** : အစမှတ်နှင့် အဆုံးမှတ် (အချိန် သို့မဟုတ် နေရာ) ကို ဖော်ပြသည်။ 


5. 
**N1 と N2** : နာမ်နှစ်ခုကို ချိတ်ဆက်သည်။

### **Chapter 5.txt**

**ဝေါဟာရများ (Vocabulary)** 

* いきます (行きます) : သွားသည် 


* きます (来ます) : လာသည် 


* かえります (帰ります) : ပြန်သည် 


* がっこう (学校) : ကျောင်း 


* えき (駅) : ဘူတာ 


* ひこうき (飛行機) : လေယာဉ်ပျံ 


* でんしゃ (電車) : ရထား 


* じてんしゃ (自転車) : စက်ဘီး 


* あるいて (歩いて) : လမ်းလျှောက်ပြီး 


* ともだち (友達) : သူငယ်ချင်း 


* かぞく (家族) : မိသားစု 


* いつ : ဘယ်အချိန် 


* たんじょうび (誕生日) : မွေးနေ့ 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. 
**N (နေရာ) へ 行きます/来ます/帰ります** : သွားလာရာ ရှေးရှုရာအရပ်ကို ပြသည်။ 


2. 
**どこ [へ] も 行きません** : ဘယ်ကိုမှ မသွားပါ (လုံးဝအငြင်းပုံစံ)။ 


3. 
**N (ယာဉ်) で 行きます** : သွားလာသည့် နည်းလမ်း/ယာဉ်ကို ပြသည်။ 


4. 
**N (လူ/တိရစ္ဆာန်) と V** : အတူတူ လုပ်ဆောင်သူကို ပြသည်။ 


5. 
**いつ** : အချိန်ကို မေးမြန်းရာတွင် သုံးပြီး "ni" ဝိဘတ် မလိုက်ရပါ။

### **Chapter 6.txt**

**ဝေါဟာရများ (Vocabulary)** 

* たべます (食べます) : စားသည် 


* のみます (飲みます) : သောက်သည် 


* みます (見ます) : ကြည့်သည် 


* ききます (聞きます) : နားထောင်သည် 


* よみます (読みます) : ဖတ်သည် 


* かきます (書きます) : ရေးသည် 


* かいます (買います) : ဝယ်သည် 


* とります (撮ります) : ရိုက်သည် (ဓာတ်ပုံ) 


* します : ပြုလုပ်သည် 


* ごはん : ထမင်း 


* おちゃ (お茶) : လက်ဖက်ရည်ကြမ်း 


* なに (何) : ဘာ 


* いっしょに : အတူတူ 



**သဒ္ဒါရှင်းလင်းချက် (Grammar Notes)** 

1. 
**N を V (အပြုခံကြိယာ)** : ကံပုဒ်ကို "wo" ဝိဘတ်ဖြင့် ပြသည်။ 


2. 
**何 を しますか** : ဘာလုပ်မှာလဲဟု မေးမြန်းသည်။ 


3. 
**N (နေရာ) で V** : အပြုအမူ လုပ်ဆောင်သည့် နေရာကို ပြသည်။ 


4. 
**Vませんか** : ဖိတ်ခေါ်ခြင်း (အတူတူ လုပ်ကြမလား)။ 


5. 
**Vましょう** : အဆိုပြုခြင်း (လုပ်ကြရအောင်)။
